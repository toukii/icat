# ISVG

svg to image(png)

## usage


 - __DecodeSVG__
 
```
bs := goutils.ReadFile("github.svg")

dbs, err := DecodeSVG(bs)
if err != nil {
	t.Errorf("%s", err)
}
if err := icat.ICatRead(bytes.NewReader(dbs), os.Stdout); err != nil {
	t.Errorf("%s", err)
}
```

 - __DisplaySVG__

```
if err := DisplaySVG(bs); err != nil {
	t.Errorf("%s", err)
}
```
